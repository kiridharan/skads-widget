# Changelog

## 0.1.0

- 🎨 **Token System** — `SkadTokens` ThemeExtension with light/dark presets (zinc palette)
- 🏗 **Theme Helper** — `SkadTheme.light()` / `SkadTheme.dark()` for quick setup
- 🎬 **Motion System** — `SkadMotion` animation duration and curve tokens
- 🔘 **SkadButton** — 5 variants (primary, secondary, outline, ghost, destructive), 4 sizes, loading state, icon slots, keyboard/hover/focus support
- 📝 **SkadInput** — Label, error, helper text, prefix/suffix, focus ring, disabled state
- 💬 **SkadDialog** — Animated overlay, title/description/actions, keyboard dismiss
- 🖥 **CLI Tool** — `skad init` and `skad add <widget>` for copy-first workflow
